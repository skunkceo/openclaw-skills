# Docker Manager

Manage Docker containers, images, and compose stacks from your AI assistant.

## What It Does

- List and inspect running containers
- Start, stop, and restart containers
- View container logs
- Manage Docker Compose stacks
- Pull and manage images
- Monitor resource usage

## Installation

```bash
clawhub install docker-manager
```

## Requirements

- **Docker** installed and running
- User in the `docker` group (or use sudo)

## Usage

Ask naturally:

- "List all running containers"
- "Show logs for the nginx container"
- "Restart the wordpress stack"
- "What's using the most memory?"
- "Pull the latest postgres image"
- "Stop all containers"
