# Slack Bot

Send messages, manage channels, and automate Slack workflows.

## What It Does

- Send messages to channels and DMs
- React to messages with emoji
- Pin and unpin important messages
- Search message history
- Manage channel membership

## Installation

```bash
skunk install slack-bot
```

## Requirements

- Slack workspace with OpenClaw connected
- Appropriate bot permissions

## Setup

Slack is typically configured during OpenClaw onboarding. If not:

1. Create a Slack app at api.slack.com
2. Add required bot scopes
3. Install to your workspace
4. Add bot token to OpenClaw config

## Usage

Ask naturally:

- "Post an update to #dev"
- "React with a checkmark to Sam's last message"
- "Pin the deployment notes in #releases"
- "Search for messages about the API bug"
- "Send a DM to the team about the outage"
