# Calendar Sync

Connect and sync calendar events with your AI assistant.

## What It Does

- Read upcoming events from Google Calendar
- Create, update, and delete events
- Check availability and find free slots
- Send meeting invites
- Set reminders for important events

## Installation

```bash
skunk install calendar-sync
```

## Requirements

- Google Calendar API credentials (service account or OAuth)

## Setup

1. Create a Google Cloud project
2. Enable the Google Calendar API
3. Create credentials (service account recommended)
4. Share your calendar with the service account email
5. Add credentials path to your OpenClaw config

## Usage

Ask naturally:

- "What's on my calendar tomorrow?"
- "Schedule a meeting with Sam at 3pm Friday"
- "Find a free 30-minute slot this week"
- "Cancel my 2pm appointment"
- "Remind me about the standup in 15 minutes"
