# Twitter Monitor

Monitor Twitter/X for mentions, keywords, and industry trends.

## What It Does

- Track mentions of your brand or products
- Monitor competitor activity
- Follow industry keywords and hashtags
- Alert on important tweets
- Analyze engagement patterns

## Installation

```bash
skunk install twitter-monitor
```

## Requirements

- Twitter/X API credentials (OAuth 1.0a or 2.0)
- API access level appropriate for your needs

## Setup

Add Twitter API credentials to OpenClaw config or environment:

```bash
export TWITTER_API_KEY="xxx"
export TWITTER_API_SECRET="xxx"
export TWITTER_ACCESS_TOKEN="xxx"
export TWITTER_ACCESS_SECRET="xxx"
```

## Usage

Ask naturally:

- "Check for mentions of SkunkCRM today"
- "What are people saying about WordPress CRM?"
- "Monitor tweets about our competitor"
- "Find trending topics in the WordPress space"
- "Alert me when someone mentions our plugin"
