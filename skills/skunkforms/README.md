# SkunkForms Skill

Manage SkunkForms forms and submissions via WP-CLI.

## Features

- **Forms**: List, get, create, update, delete forms
- **Submissions**: View and export form submissions
- **Webhooks** (Pro): Trigger webhooks for submissions

## Requirements

- WordPress site with SkunkForms installed
- WP-CLI access (local via Studio, or remote via SSH)
- SkunkForms Pro for create, update, delete, and webhook operations

## Installation

```bash
skunk install skunkforms
```
