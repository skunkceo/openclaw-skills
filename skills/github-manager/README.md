# GitHub Manager

Manage repositories, pull requests, issues, and GitHub Actions from your AI assistant.

## What It Does

- Clone, create, and manage repositories
- Create and review pull requests
- Manage issues and labels
- Monitor and trigger GitHub Actions
- Search code and commits

## Installation

```bash
clawhub install github-manager
```

## Requirements

- **GitHub CLI (gh)** installed and authenticated

```bash
# Install gh CLI
brew install gh  # macOS
# or: sudo apt install gh  # Ubuntu

# Authenticate
gh auth login
```

## Usage

Ask naturally:

- "Create a PR from my current branch to main"
- "List open issues in skunkcrm-pro"
- "Clone the openclaw-skills repo"
- "Show me the latest failed workflow runs"
- "Create an issue for the bug we discussed"
