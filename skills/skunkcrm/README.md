# SkunkCRM Skill

Manage SkunkCRM contacts, activities, deals, automations, and email campaigns via WP-CLI.

## Features

- **Contacts**: List, get, create, update contacts with tags and custom fields
- **Activities**: View activity timeline, add notes to contacts
- **Deals** (Pro): Manage sales pipeline, create and update deals
- **Automations** (Pro): List and trigger automation workflows
- **Campaigns** (Pro): View email campaign stats and analytics
- **Reports** (Pro): Generate CRM reports

## Requirements

- WordPress site with SkunkCRM installed
- WP-CLI access (local via Studio, or remote via SSH)
- SkunkCRM Pro for deals, automations, campaigns, and reports

## Installation

```bash
skunk install skunkcrm
```
