# WP Content Manager

Create, edit, and manage WordPress posts, pages, and media.

## What It Does

- Create and update posts and pages
- Manage categories and tags
- Upload and organize media
- Schedule content publication
- Bulk edit content metadata

## Installation

```bash
skunk install wp-content-manager
```

## Requirements

- **WP-CLI** installed on the WordPress server
- Or WordPress REST API credentials

## Usage

Ask naturally:

- "Create a new blog post draft"
- "Publish the post about CRM features"
- "Add the 'tutorial' tag to recent posts"
- "Schedule this post for Monday 9am"
- "Upload an image and add it to the post"
- "List all draft posts"
