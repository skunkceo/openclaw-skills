# Social Scheduler

Schedule and manage social media posts across platforms.

## What It Does

- Schedule posts for Twitter/X, LinkedIn, Facebook
- Queue content for optimal posting times
- Track scheduled post status
- Repurpose content across platforms
- Manage posting calendar

## Installation

```bash
skunk install social-scheduler
```

## Requirements

- API credentials for target platforms:
  - Twitter/X API keys
  - LinkedIn API access
  - Facebook/Meta API access

## Setup

Configure platform credentials in OpenClaw config or environment variables.

## Usage

Ask naturally:

- "Schedule a tweet for 9am tomorrow"
- "Post this to LinkedIn and Twitter"
- "What's scheduled for this week?"
- "Cancel the post scheduled for Friday"
- "Find the best time to post today"
