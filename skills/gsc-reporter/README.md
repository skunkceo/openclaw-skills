# GSC Reporter

Pull Google Search Console data and generate SEO performance reports.

## What It Does

- Fetch search performance data (clicks, impressions, CTR, position)
- Track keyword rankings over time
- Monitor indexing status
- Identify top performing pages
- Spot ranking changes and opportunities

## Installation

```bash
clawhub install gsc-reporter
```

## Requirements

- Google Search Console API access
- Service account with GSC property access

## Setup

1. Create a Google Cloud service account
2. Add the service account email as a user in GSC
3. Configure the credentials path in OpenClaw

## Usage

Ask naturally:

- "Show me top keywords for skunkcrm.com"
- "What pages got the most clicks this week?"
- "Which keywords dropped in ranking?"
- "Generate a weekly SEO report"
- "Check indexing status for our blog posts"
