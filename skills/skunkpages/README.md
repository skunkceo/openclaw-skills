# SkunkPages Skill

Manage SkunkPages landing pages and templates via WP-CLI.

## Features

- **Pages**: List, get, create, update, delete, duplicate landing pages
- **Templates**: Browse and use landing page templates
- **Publishing** (Pro): Publish or schedule pages for future publication

## Requirements

- WordPress site with SkunkPages installed
- WP-CLI access (local via Studio, or remote via SSH)
- SkunkPages Pro for create, update, delete, duplicate, and publish operations

## Installation

```bash
skunk install skunkpages
```
