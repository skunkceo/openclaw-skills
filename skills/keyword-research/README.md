# Keyword Research

Research keywords and analyze search intent for content planning.

## What It Does

- Find related keywords and phrases
- Analyze search volume estimates
- Assess keyword difficulty
- Identify content gaps
- Suggest long-tail variations
- Analyze competitor keywords

## Installation

```bash
clawhub install keyword-research
```

## Requirements

None required. Enhanced data available with:
- Google Search Console access
- Ahrefs/SEMrush API (optional)

## Usage

Ask naturally:

- "Research keywords for 'wordpress crm plugin'"
- "Find long-tail keywords around form builders"
- "What questions do people ask about CRM?"
- "Analyze keyword difficulty for 'free crm'"
- "What keywords are competitors ranking for?"
